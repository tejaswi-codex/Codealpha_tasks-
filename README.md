
# CodeAlpha Internship Tasks - Python Projects

## 📌 Overview
This repository contains Python mini-projects developed as part of the CodeAlpha Internship Program. The projects demonstrate beginner-level programming skills, problem-solving ability, and understanding of core Python concepts.


## 💻 Technologies Used
- Python 
- Conditional Statements (if-else)
- Loops (for, while)
- Functions
- Random Module
- Data Structures (Lists, Dictionaries, Strings)
- Arithmetic Operations 
- File Handling (.txt)
- User Input/Output

## 📁 Projects Included

### 1. Basic Chatbot
A simple rule-based chatbot built using Python.

**Features:**
- Responds to user inputs using predefined rules
- Uses conditional logic
- Beginner-level conversational flow

---

### 2. Hangman Game
A word guessing game where the user must guess the correct word within limited attempts.

**Features:**
- Random word selection
- Limited attempts system
- Tracks correct and wrong guesses
- Interactive console-based gameplay

---

### 3. Stock Portfolio Tracker
A simple Python project that calculates the total
investment value based on user-entered stock quantities and predefined stock prices.

**Features:**
- User enters stock name and quantity
- Uses a hard-coded dictionary for stock prices
- Calculates total investment value
- Saves results to a text file


## ⚙️ How to Run
1. Install Python 3
2. Download or clone this repository.
git clone https://github.com/tejaswi-codex/Codealpha_tasks-.git
3. Open the project file you want to run
